Type of loop in Java:-

1 Entry Control loop:-

First Check Condition than execute statement.

1.1 while :-

this loop can be implemented for finite condition and infinite condition both.

but we mostly prefer while loop for infinite condition.

Syntax:-
Finite Loop:-
init;
while(condition)
{
    statement;
    increment;
}


InFinite Loop

while(true)
{
    if(condition)
       break;

}

1.2 for:-

  it is used to create program where condition is finite.
          1       2 5             4 7
   for(init;condition;iteration)
      {
                3 6
               statement;
      }

i++;
++i;
 
   for(int i=1;i<5;++i)
    {
        System.out.println(i);
    }

 for(int i=1;i<5;i++)
    {
        System.out.println(i);
    }





1.3  foreach :-

it is used to display array and collection elements.



2 Exit Control loop:-

First Execute Statement under Loop than check condition

init:

    statement;
